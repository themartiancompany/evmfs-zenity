=================
evmfs-get-zenity
=================

---------------------------------------------------------
Ethereum Virtual Machine File System file downloader
---------------------------------------------------------
:Version: evmfs-get-zenity |version|
:Manual section: 1

Synopsis
========

evmfs-get-zenity *[options]*

Description
===========

Simple UI written in Bash with Zenity used
to retrieve files from the
Ethereum Virtual Machine File System (EVMFS).

Options
=======

-h                      Displays help message.
-c                      Enable color output
-v                      Enable verbose output
